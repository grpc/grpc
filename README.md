# google-grpc


Generating grpc client and server code in dlang for [hunt-grpc](https://github.com/huntlabs/hunt-grpc)

protoc -I ./examples --grpc_out=./examples --plugin=protoc-gen-grpc=grpc_dlang_plugin ./examples/helloworld.proto
