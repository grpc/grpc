SKIP_AUTOMOC¶

New in version 3.8.

Exclude the source file from AUTOMOC processing (for Qt projects).

For broader exclusion control see SKIP_AUTOGEN.

EXAMPLE¶

# ...
set_property(SOURCE file.h PROPERTY SKIP_AUTOMOC ON)
# ...
