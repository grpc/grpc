Module: src/server

Server module This module contains all the server code for Node gRPC: both the Server class itself and the method handler code for all types of methods. For example, to create a Server, add a service, and start it: var server = new server_module.Server(); server.addProtoService(protobuf_service_descriptor, service_implementation); server.bind('address:port', server_credential); server.start();
