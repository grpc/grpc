Detailed Description

A struct containing all information necessary to schedule/cancel a server authorization check request.

cb and cb_user_data represent a gRPC-provided callback and an argument passed to it. result will store the result of server authorization check. target_name is the name of an endpoint the channel is connecting to and certificate represents a complete certificate chain including both signing and leaf certificates. status and error_details contain information about errors occurred when a server authorization check request is scheduled/cancelled. It is used for experimental purpose for now and subject to change.
