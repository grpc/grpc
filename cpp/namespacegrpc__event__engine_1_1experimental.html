std::shared_ptr<EventEngine> grpc_event_engine::experimental::GetDefaultEventEngine ( )

Lazily instantiate and return a default global EventEngine instance if no custom instance is provided.

If a custom EventEngine is provided for every channel/server via ChannelArgs, this method should never be called, and the default instance will never be instantiated.
